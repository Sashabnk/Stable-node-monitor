# Stable-node-monitor
Telegram bot for monitoring Stable Testnet nodes. Features multi-node support, real-time status checks, and a simple 1-line installer.

```bash
wget https://raw.githubusercontent.com/Sashabnk/Stable-node-monitor/main/install.sh && chmod +x install.sh && ./install.sh
```

To ensure that this bot works efficiently and without complaints, the node should be set up according to these instructions: https://sashadrop.short.gy/stable-node-en
